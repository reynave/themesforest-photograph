HTML Structure - top

CSS Framework with Getbootstrap v5.3.0

We are using getbootstrap v5.3.0 as the basic HTML framework. If you want to dismantle a basic HTML framework, I highly recommend understanding the working principle of getbootstrap ver 5.3.0

This theme is a flexible layout with two columns. The sidebar's (1st col ) content is within a div with an id of "menu-left" inside col-2. All of the information within the main content area is nested within a div with an id of "col-md-10 col-12" or greatest col. The general template structure is the same throughout the template. Here is the general structure.

CSS and SASS Files - top

I'm using only one SASS file in this theme. All custom variables are in sass in the initial line 

/**
* SASS Code ver 1.0
*/

/* Import google font */
@import url('https://fonts.googleapis.com/css2?family=Nunito+Sans:wght@300&display=swap')
@import url('https://fonts.googleapis.com/css2?family=Manrope:wght@300&display=swap')
@import url('https://fonts.googleapis.com/css2?family=Red+Hat+Display:wght@300;400;700&display=swap')

/* Global Varibale */
$font-times: "Times New Roman", Times, serif
$primary-color: #333
$black-color: #000
$black-color-2: #394149
$white-color: #fff
$animation-duration: 0.3s
$font-nunito: 'Nunito Sans', sans-serif
$font-manrope: 'Manrope', sans-serif
$font-red-hat: 'Red Hat Display', sans-serif

/* your custome style here */

We don't have a special sass code for this theme, all "div" structures use getbootstrap

If there are additions or changes to the mobile version, you can add the code as follows

/* SASS Code */
@media only screen and (max-width: 540px) 
	add styles here

/* CSS Code */
@media only screen and (max-width: 540px){
	add styles here
}

JavaScript - top

This theme imports Javascript files.

  1. jQuery
    jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. bootstrap.bundle.min.js and bootstrap.min.css
    Doc here : getbootstrap.com/docs/5.3/getting-started/introduction/
  3. Bootstrap Icon
    Link here : icons.getbootstrap.com
  4. Slick-1.8.1 for Slider / Banner
    Fully responsive. Scales with its container, extremely simple and fast.
    Doc here : kenwheeler.github.io/slick/
  5. Masonry Layout
    you have to put both plugin together
    masonry.pkgd.min.js
    imagesloaded.pkgd.min.js
  6. Magnific Popup - v1.1.0
    Responsive lightbox & dialog script with focus on performance and providing best experience for user with any device (for jQuery or Zepto.js).
    Doc here : dimsemenov.com/plugins/magnific-popup/
  7. Animate.css for text animated
    Doc here : animate.style
  8. I'm use CSS3 Full Screen Slideshow for Home 2
    Sample here : https://tympanus.net/Tutorials/CSS3FullscreenSlideshow/
    Code here : codepen
    Doc here : Fullscreen Background Image Slideshow with CSS3
    this is not include in package.json
  9. livereload.js For development only
    NPM www.npmjs.com/package/livereload
    An implementation of the LiveReload server in Node.js.
    <script src="http://localhost:35729/livereload.js"></script>
I'm save all package manager for the JavaScript in package.json, if you are familiar with nodejs, just npm install donwload all dependencies
